Formulario carga de Alumnos

Nombre
El campo nombre es requerido La cantidad mínima del campo nombre es de 3 caracteres.

Apellido
El campo nombre es requerido La cantidad mínima del campo nombre es de 3 caracteres.

Fecha de Nacimiento
El campo fecha de Nacimiento es requerido El campo fecha no cumple con el formato dd/mm/yyyy

El campo mail es requerido No cumple con el formato de mail, ejemplo a@gmail.com

Direccion
El campo direccion es requerido La cantidad mínima del campo direccion es de 3 caracteres.

Teléfono
El campo telefono es requerido El campo fecha no cumple con el formato numerico
